# README

This is a very simple web application created for educational purposes: a Todo app written in Ruby on Rails and React.

## Dependencies:
- Ruby 3.2.2
- Node.js 20
- Yarn
- SQLite

## Before the first run
`bundle install`
`bundle exec rake db:setup`
`yarn install`

## Run in development mode
`bin/dev`

## Run tests
`bundle exec rake`
